# Theoria

Notebooks to learn:

- **Python**
- **Statistics**
- **Computer Science**
- **Mathematics**
- **Finance**

Notebooks should be labeled as follows:

- **Basic**: For beginners, covering the essentials.
- **Intermediate**: For those with some experience, ready to explore more.
- **Advanced**: For practitioners or those with a deeper interest in the subject.
- **Examples**: See the concepts in action with practical applications.
